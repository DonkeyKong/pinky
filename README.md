# pinky
_A picture frame that takes pictures_

## Required Hardware
- Raspberry Pi Pico or Pico 2
- Arducam Mega SPI camera
- Pimoroni Inky e-ink display
  - Inky wHAT 4.2" (B/W or 3 Color) (SSD1683)
  - Inky Impression 5.7" (UC8159)
  - Inky Impression Spectra 7.3" (E673)
  - Easily expanded support to other models

## Optional Hardware
- Lithium polymer battery + charging circuit
- Push buttons
- Power switch
